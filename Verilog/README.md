## Verilog Version of MMU09

This folder has the Verilog code for the CPLD as a Quartus II project.
It also has a Verilog implementation of the 6809, ROM, RAM and a UART.

The files `mc6809e.v` and `mc6809i.v` come from Greg Miller cycle-accurate
implementation of the 6809 in Verilog at https://github.com/cavnex/mc6809

Tim Rudy wrote the testbed helper file `tbhelper.v`.
